The system will also match partial names, so these work too:
- adeola.jpg (for Adeola Ojukwu)
- ibrahim.jpg (for Ibrahim Mohammed)
- funke.jpg (for Funke Adebayo)

## Supported Formats
- JPG, JPEG, PNG, WebP
- Recommended size: 200x200px
- Max file size: 1MB per image
